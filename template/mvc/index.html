# MVC 项目样板

MVC项目样板是一个标准的、传统的MVC开发模式的项目骨架，模型简单易理解 非常适用于开发普通的企业网站

# 特点

  • 视图是支持所有 cordinate 支持的所有模板引擎
  • 视图 render 函数本身也是个插件，你可以自由定义它
